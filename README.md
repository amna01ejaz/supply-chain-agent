# 📦 Enterprise Multi-Agent Supply Chain & Inventory Optimization Simulator

A professional, stateful operations modeling and predictive replenishment application engineered using **Python**, the **Agno Multi-Agent Framework**, **Pandas**, and **Plotly**, powered by **Google Gemini 2.5 Flash** models and wrapped in an interactive **Streamlit** architecture.

This workspace simulates real-world distribution center challenges by using cooperative AI agents to clean conversational metrics log segments, compute stock burn rates, and execute localized reorder formulas to prevent system stockouts.

## ✨ Architectural Core Features
* **Asynchronous Multi-Agent State Handoff:** Establishes a modular two-agent team layout where data parsing layers pass structured context frames directly to procurement calculation nodes.
* **Algorithmic Depletion Analytics:** Automatically deduces quantitative risk variables (Days of Remaining Stock, Supplier Lead Time Points, and Minimum Safety Stock Levels).
* **Automated B2B Purchase Orders:** Instructs the predictive reasoning module to output a structured procurement matrix indicating optimal 30-day balance orders.
* **Responsive Visual Performance Grid:** Integrates hardware-accelerated Plotly Express bar distributions color-coded by situational alert tiers (Stable vs. Critical Risk).

## 🛠️ Technology Stack
* **Agent Operations Controller:** Agno Framework Team Topologies
* **Core Language Model Core:** Google Gemini 2.5 Flash API
* **Mathematical Vector Analytics:** Pandas, NumPy
* **Data Visualization Layers:** Plotly Express
* **Application Layout Server:** Streamlit User Interface Engine
